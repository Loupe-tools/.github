### Why Loupe?

SOC analysts, incident responders, and security-conscious users need a way to safely inspect suspicious files — without uploading them to third-party services or spinning up a sandbox. Loupe runs entirely in your browser; **nothing ever leaves your machine**.

- 🔒 **Zero network access** — strict CSP blocks all external requests
- 📄 **Single HTML file** — no install, no dependencies, works on any OS
- 🧬 **60+ file formats** — Office, PDF, email, archives, PE/ELF/Mach-O, scripts, images, certificates, and more
- 🎯 **Built-in YARA engine** — in-browser rule matching with 200+ default detection rules
- 🔎 **IOC extraction** — URLs, IPs, email addresses, file paths, and encoded content auto-detected
- 🧩 **VBA / macro analysis** — extracts, decodes, and syntax-highlights embedded macros
- 🌙 **Midnight Glass theme** — polished dark mode with light theme toggle
